### Clone ###
```
git@github.com:payeworks/paye_project_frontend.git paye_project_frontend
```

### Required ###
* NodeJS
* GulpJS
* Bower

### Install ###

```
#!bash

$ cd paye_project_frontend

$ npm i
```

### Compile ###

```
#!bash

$ npm install -g gulp
or
$ sudo npm install -g gulp
```

### Test ###

```
#!bash

$ gulp test
```

### Run for development ###

```
#!bash

$ node server.js
```

### Test ###
Open http://localhost:8080
